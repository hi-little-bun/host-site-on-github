Step-by-Step Guide to Hosting a Site on GitHub

Step 1: Create a GitHub Repository

Log in to your GitHub account and click on the "New" button to create a new repository. Name your repository appropriately, and ensure it is set to public.

Step 2: Add Your Website Files

Upload your HTML, CSS, JavaScript, and other necessary files to the repository. You can do this by dragging and dropping files into the upload area or using Git commands.

Step 3: Enable GitHub Pages

Go to the repository settings, scroll down to the "Pages" section, and select the branch and directory for your site (e.g., main branch and root directory). Click "Save" to publish your site.

Step 4: Access Your Website

GitHub will provide a link to your live website. It typically follows the format: https://username.github.io/repository-name/. Open this link to view your hosted site.
